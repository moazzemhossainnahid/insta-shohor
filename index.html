Question and Answer

JavaScript is a client-side scripting Language. It is the most powerfull scripting Language in this time. Javascript can Run in Web Browser with JavaScript Engine. Google chrome has their own JavaScript Engine Called V8. Different Browser use Different Engine for Run JavaScript. for Example: Microsoft Edge Use 'Chakra' Engine, safari use 'JavaScript Core' Engine, Firefox use 'Spidermonkey' Engine for Run JavaScript Code. we use JavaScript for any website interactive and user friendly.

JavaScript is a Single Threaded scripting Language. But Browser APIs work as separate threade. that's why JavaScript use event loop as setTimeout. event loop work secret behind JavaScript asynchronous programming. it checks whether or not the call stack is empty. If empty, new functions are added from the event queue. If it's not, then the current function call is processed.

Local Storage
Local Storage is a Storage that set data from user in Browser. Local Storage can read only JSON formatted code. Local Storage Save your data after browser closed and then after any time when you re open that website Local Storage can show your old data. it useally work when user login a website.

Session Storage
Session Storage is a Storage that set data from user in Browser. Session Storage can read only JSON formatted code. Session Storage Save your data when you run a website. after when you close your browser, Session Storage Deleted Your Data. then when you reopen that website session storage dont give you any data.
